Разворачивающиеся текстовые блоки

Этот код можно встроить в любую часть вашей страницы. Скопируйте HTML-структуру аккордеона, стили CSS и JavaScript код.

Кликните на заголовок, чтобы развернуть или свернуть текстовый блок.

Что такое JavaScript?
▼

JavaScript — это язык программирования, который используется для создания интерактивных веб-страниц. Он позволяет добавлять динамическое поведение, обрабатывать события, управлять DOM и многое другое.

JavaScript является одним из трех основных языков веб-разработки, наряду с HTML и CSS.

Как работает этот аккордеон?
▼

Аккордеон работает на чистом JavaScript. При клике на заголовок:

  • Находится родительский элемент с классом accordion-item
  • Переключается класс active на этом элементе
  • Изменяется максимальная высота max-height для содержимого
  • Поворачивается иконка для визуальной индикации

При этом можно настроить так, чтобы при открытии одного элемента другие автоматически закрывались, или разрешить одновременное открытие нескольких элементов.

Как встроить этот код на свою страницу?
▼

Чтобы использовать этот аккордеон на своей странице:

  1. Скопируйте HTML-структуру аккордеона (элементы с классами accordion, accordion-item, accordion-header и accordion-content)
  2. Добавьте CSS стили в раздел стилей вашей страницы или во внешний CSS файл
  3. Добавьте JavaScript код в конец вашей страницы перед закрывающим тегом </body> или во внешний JS файл
  4. Измените содержимое блоков под свои нужды
Дополнительные возможности
▼

Вы можете легко настроить этот аккордеон под свои нужды:

  • Анимации: Измените параметры transition в CSS для другой скорости анимации
  • Внешний вид: Измените цвета, отступы, шрифты в CSS-стилях
  • Поведение: В JavaScript можно изменить логику работы (открывать только один элемент одновременно или разрешить несколько открытых)
  • Иконки: Вместо символа ▼ можно использовать SVG-иконки или шрифтовые иконки

Код написан максимально просто и не требует внешних библиотек, поэтому его легко модифицировать.
